# **onigiri**

**onigiri** is a native-styled, flat Anki flashcard theme based on [Prettify](https://github.com/pranavdeshai/anki-prettify) designed for [Onigiri](https://github.com/thepeacemonk/Onigiri)

---

### **Core Features**

* **Flat Aesthetic**: Removed all drop shadows and stylized gradients in favor of a clean, "border-only" look.
* **Native Integration**: Dynamically inherits Anki's window background and text colors to ensure perfect visual integration in both light and dark modes.
* **System Typography**: Uses the native OS font stack (San Francisco on macOS, Segoe UI on Windows) for consistent system-wide readability.
* **Pill-Shaped Tags**: Automatically transforms hierarchical tags into modern, neutral-colored pill shapes.
* **Deck Breadcrumbs**: Displays your current deck hierarchy at the top of each card for quick context.
* **Responsive**: Fully optimized for Anki Desktop, AnkiMobile (iOS), and AnkiDroid.

---

### **Installation & Usage**

1.  **Direct Download**: Download the master package.
2.  **Import**: Open Anki and select **Import File** (or `⋮` -> `Import` on AnkiDroid).
3.  **Note Types**: The package will automatically create the **Onigiri Basic**, **Onigiri Basic (and reversed card)**, and **Onigiri Cloze** note types.
